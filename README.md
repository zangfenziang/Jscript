# Jscript
描述
```js
jww发明了一种神奇的语言，叫Jscript
这种语言全部由一个一个函数组成的，并且所有参数都是整数
其中jww定义了一个初始函数，在c，c++中，可以看作如下形式
void print(int x){
    printf("%d\n", x);
}
在Jscript中每一个函数的定义都如同下面这个函数一样:
var cjb = (x,y)=>{
    print(x);
    jww(y);
}
其中cjb是函数名，这个函数需要有两个参数(保证函数名和参数名都是小写字母)。调用这个函数时，会输出x并且调用jww(y)这个函数
在每一个Jscript文件中，每个函数的位置是无所谓的，并且一定会定义有一个函数main。在之后运行这个文件的时候，系统会默认调用main(0)
```
输入
```
每一个输入第一行是一个正整数t
之后有t个Jscript文件
每个文件的结尾都有单独一行的#结束
```
输出
```
对于每一个Jscript文件，请输出他输出的所有数的和对131411的模数
如果出现无法停止运行的Jscript文件，请输出NaN
```
数据范围
```
对每一个输入，所有函数个数小于100000，所有函数被其他函数引用的次数和小于1000000
```
样例
```js
1
var cjb = (x,y)=>{
    print(x);
    jww(y);
}
var jww = (x)=>{
    print(5);
}
var main = (x)=>{
    cjb(3, 7);
}
```
